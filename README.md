# 今天吃什麼？ - 美食決策助手

一個 Progressive Web App (PWA) 讓您隨機選擇今天要吃什麼，支援自訂食物清單和分享功能。

## 功能特色

- 🎯 **隨機選擇** - 根據食物類型隨機推薦
- 📱 **PWA 支援** - 可安裝到手機主畫面
- 💾 **離線使用** - 使用 IndexedDB 儲存資料
- 🛠️ **自訂清單** - 新增、刪除個人喜好的食物
- 🔗 **分享功能** - 分享 App 給朋友使用
- 📱 **響應式設計** - 適配各種手機尺寸

## 安裝說明

### 方法 1: 直接訪問
1. 用手機瀏覽器打開連結
2. 點擊分享頁面的「安裝App」按鈕
3. 按照提示添加到主畫面

### 方法 2: 開發模式
1. 用 VSCode 打開專案
2. 安裝 Live Server 擴展
3. 右鍵點擊 `index.html` 選擇「Open with Live Server」
4. 用手機訪問顯示的本地網址

## 技術架構

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **儲存**: IndexedDB
- **PWA**: Service Worker, Web App Manifest
- **圖示**: Font Awesome
- **QR Code**: qrcode.js
